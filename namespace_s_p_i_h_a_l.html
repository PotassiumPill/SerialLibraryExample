Detailed Description

SPI HAL global namespace.

This namespace contains all global functions and definitions needed to implement low level drivers for SPI serial communication.

Enumeration Type Documentation

◆ ClockMode

enum SPIHAL::ClockMode
strong

An enum type for the SPI transfer mode.

Enumerator
Mode0 

Data is sampled on rising leading edge, data is changed on falling trailing edge.

Mode1 

Data is changed on rising leading edge, data is sampled on falling trailing edge.

Mode2 

Data is sampled on falling leading edge, data is changed on rising trailing edge.

Mode3 

Data is changed on falling leading edge, data is sampled on rising trailing edge.

void SPIHAL::InitSercom ( SPIHAL::Peripheral *  peripheral,
bool  is_client = false 
)

Initializing function for SPI serial communication low level drivers.

Initializes the hardware to use SPI, setting the I/O multiplexing, baud rate, feeding clocks, enabling interrupts, and enabling SERCOM MISO and MOSI data. This SPI HAL sets the hardware into master/host mode.

Parameters
peripheralpointer to peripheral information struct
is_clientconfigures SPI peripheral as client if set true (default = false)
