Detailed Description

Serial Buffer object

This is a Serial Buffer object which acts as a generic high-level character array with FIFO functionality and basic string reading abilities.

Reads entire buffer and checks if it is equal to the input string + numeric parameter.

Checks input string with front of buffer with offset [1, max_digits], and then returns the ASCII number characters following string as unsigned int.
Example: GetIntParam("c_") will set output pointer value to unsigned int 12u if "c_12" was in the front of the buffer.
Can also be used multiple times for different strings with parameters and in conjunction with GetString() calls.
If a string is detected, all calls will be disabled until a new data is added to buffer.
delimiter parameter will detect an end-character if set to nonzero to allow for more secure message reading.
max_digits parameter will set a bound for the max amount of digits in the parameter. If max_digits > 9, the return value may not be what is expected.

Parameters
outputpointer to received int value
inputinput string to be detected
max_digitsmaximum number of digits in parameter, must not exceed 9
delimiteroptional delimiting character after int parameter (default = '\0')
int_funcinterrupt function pointer to be invoked
Returns
success of reception (will return false if string value not detected, no integer detected, or no delimiter detected if set)
Note
Input array must be null-character terminated.

Reads entire buffer and checks if it is equal to the input string.

First removes all entries in buffer.
Checks input string with front of buffer, returning true if the input string has been detected.
Can be used multiple times for different strings.
If a string is detected, all calls will be disabled until a new character is received.
Shift parameter detects substring offset by shift places. Useful for detecting a command with parameter (i.e. "c_25").
Adds chars after found shifted string and moves the receive buffer pointer back to the end of the substring if move_pointer is set to true.
