Detailed Description

template<typename T>
class GenericBuffer::GENERIC_BUFFER< T >

Generic ring buffer template.

Template class for a generic FIFO ring buffer. Offers very basic put and get functionality, with the ability to access raw elements to directly access and alter elements.

Shifts the read pointer forwards or backwards.

Allows for direct manipulation of where the read pointer is, handling all calculations for capping shift amount and updating the current length of the buffer.

Parameters
shift_sizesize of the shift based on number of elements (ie shift_size = 1 shifts the read index 1 element forwards/backwards)
increasetrue = shifts forwards, false = shifts backwards
Note
Function will cap shift amount if it increases by more than the current length or decreases by more than the size of the array.
